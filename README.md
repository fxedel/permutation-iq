# shapiq-mcs



## Getting started

Install the Python package manager [`uv`](https://github.com/astral-sh/uv), e.g. via:

```
pipx install uv
```

Now, you can either call python scripts via `uv run main.py` or by first syncing and activating the virtual environment (venv):

```
uv sync
source .venv/bin/activate
python main.py
```
